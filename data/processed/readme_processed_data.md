## Data

The cleaned dataset contains only the following columns:

- `Date` — measurement date
- `Time` — measurement time
- `Location` — measurement location/site
- `Aquifer_type` — aquifer type
- `depth_m` — measurement depth in meters
- `ctdtemp_C` — water temperature in degrees Celsius
- `ctdsal` — salinity measurement
- `airtemp_C` — air temperature in degrees Celsius
- `Rn_Bqm3` — radon concentration in Bq/m³
