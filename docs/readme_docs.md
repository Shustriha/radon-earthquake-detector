## Documentation

This folder contains background notes and project-related documentation for the radon time-series anomaly detection project.

# Current files:

 - 28_questions_time_series_basics.pdf — personal notes summarizing key time-series concepts used in this project, including stationarity, differencing, ACF/PACF, ARIMA, lag features, forecast horizon, and feature engineering.

These notes provide theoretical background for the feature extraction and normal-radon modeling parts of the project.

External research papers and article citations should be placed separately in the references/ folder.
